# TestTask_Students-groups

Backend for test task. Frontend accessible by link : https://github.com/S-Pavlov1/TestTask_Students-Groups-front

# Setting up project

1. Create DB in MySQL and change [application.yaml](https://github.com/S-Pavlov1/TestTask_Students-groups/blob/master/src/main/resources/application.yaml) file to match your DB.
2. Build and run Backend part
3. Backend is running on port 8083. You can change it in [application](https://github.com/S-Pavlov1/TestTask_Students-groups/blob/master/src/main/java/com/testTask/Application.java)
